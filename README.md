# Seravelli — Italian Elegance 🇮🇹

Premium Taplink-style mini website for **Seravelli** brand.

![License](https://img.shields.io/badge/license-MIT-gold)
![Status](https://img.shields.io/badge/status-live-brightgreen)

## ✨ Features

- 🎨 **Premium Dark + Gold Design** — Italian luxury aesthetic
- 📱 **Fully Mobile Responsive** — optimized for all screen sizes
- 🔗 **Social Links** — Phone, Telegram, Instagram, Location
- 💾 **Save Contact** — downloads vCard (.vcf) file
- 📤 **Smart Share** — Web Share API + Telegram fallback
- 📊 **Analytics Dashboard** — password-protected admin panel
- ✨ **Micro-Animations** — shimmer, float, glow, ripple effects
- 🗄️ **Firebase Ready** — with LocalStorage fallback

## 🚀 Quick Start

1. Clone the repo
2. Open `index.html` in a browser
3. That's it! No build step needed.

## 📂 Structure

```
├── index.html        # Main page
├── style.css         # Design system & responsive styles
├── app.js            # App logic (vCard, share, tracking)
├── firebase-db.js    # Database module (Firebase/LocalStorage)
└── README.md
```

## 🔥 Firebase Setup (Optional)

1. Create a project at [Firebase Console](https://console.firebase.google.com/)
2. Enable Firestore Database
3. Update `FIREBASE_CONFIG` in `firebase-db.js`
4. Add Firebase SDK scripts to `index.html`
